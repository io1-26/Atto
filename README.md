About:

A small, keyboard-only text editor with a built-in terminal. Made for fun and experimentation.

This program was inspired by GNU Nano. There are no buttons and no mouse support. 
Everything from file navigation to text editing is done using the keyboard.

Workflow (TL;DR):

The program starts in a terminal.
You navigate using Linux-like commands (cd, ls, mkdir, etc.).
Open a file by typing: open [filename]
Edit the file.
Press Ctrl + S to save.
Press Ctrl + X to return to the terminal.
To see all available commands, type: commands
Note:

This is a simple project, not intended to be a full-featured editor. 
I built it to explore full control over text editing, rendering cells, syntax highlighting, and similar low-level concepts.
